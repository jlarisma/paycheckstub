This plugin is used to retrive username by email.

== Description ==

This plugin display form to enter email Id to retrieve forgotten username.

`Attributes`
1. settings for button lable
2. settings for success message
3. settings for mail content


== Installation ==

1. Upload from admin side, upload new plugin from Plugins -> Add New OR extract 'forget-username.zip' file in directory to your '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' in your WordPress Admin Panel.
3. To display forget username form , add `[forget-username-form]` in page,post or widget or copy and paste `<?php echo do_shortcode('[forget-username-form]');?>` in your code file.
4. To set mail formate, go to `Settings -> forget username`. Enter here your plugin settins and Update Options.
5. That's it.  Enjoy!
